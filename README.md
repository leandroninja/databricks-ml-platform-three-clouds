## Sobre o projeto

Plataforma Databricks Lakehouse multi-cloud cobrindo Azure, AWS e GCP. O projeto demonstra pipelines de dados
em camadas Bronze/Silver/Gold, treinamento de modelos ML com MLflow e pipelines de GenAI com RAG usando
Databricks Vector Search e LangChain.

## O que tem aqui

Notebooks PySpark organizados por camada (ingestão, transformação, gold, ML e GenAI), infraestrutura como
código em Terraform para as três clouds, jobs de orquestração via Databricks Workflows e scripts de bootstrap
para provisionamento inicial de cada ambiente.
